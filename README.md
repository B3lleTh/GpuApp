**StudyFlow** es una aplicación de productividad de alto nivel diseñada para optimizar sesiones de estudio mediante algoritmos de gestión de tiempo y una interfaz **Glassmorphism** de vanguardia.

---

## ✨ Características Principales

* **🛡️ AuthGate Engine:** Sistema de autenticación robusto mediante **Firebase Auth** con persistencia de estado en tiempo real.
* **💎 Glass UI Design:** Interfaz moderna con efectos de desenfoque (*BackdropFilter*), bordes con opacidad variable y estética *Deep Dark*.
* **⚙️ Algoritmos de Productividad:** Generación automática de sesiones basadas en metodologías científicas:
    * **Pomodoro:** Ciclos clásicos de 25/5 min.
    * **52-17:** El ratio de oro para la retención cognitiva.
    * **Deep Work:** Enfoque total para tareas complejas.
* **🔄 Full CRUD Real-time:** Sincronización bidireccional instantánea con **Cloud Firestore**.
* **⏱️ Animated Timer:** Cronómetro inteligente integrado por sesión con lógica de `Timer.periodic`.

---

## 🛠️ Stack Tecnológico

* **Frontend:** Flutter SDK (Material 3).
* **Backend:** Firebase Core & Firebase Auth.
* **Base de Datos:** Cloud Firestore (NoSQL).
* **Diseño:** Glassmorphism, Custom Gradients, UI Blur Effects.
